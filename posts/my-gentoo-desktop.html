My Gentoo Linux desktop

March 17, 2021

(Click here for a full sized screenshot)

I recently switched from Debian GNU/Linux to Gentoo Linux and got a new ultrawide monitor, so I wanted to share what my new desktop looks like.

Color Theme: Dracula
Window Manager: bspwm
Status Bar: polybar
Firefox CSS: Custom, based on minimal-function-fox. nightTab addon
Fonts: Ubuntu Mono (Terminal), Ubuntu Condensed (Polybar)
Folder Icons: Papirus
Terminal: Alacritty
Programs: Firefox, fet.sh (fetch tool), VS Code, ncmpcpp (music player), vim, PCManFM (file manager)

You can find my configuration files for this setup on my GitLab here.
